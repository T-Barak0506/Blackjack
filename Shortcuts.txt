MAC OS


------ NAVIGATION: ----- 

⌘T - Show all Symbols
⌃G - Go to Line...
⌘P - Go to File...
⇧⌘O - Go to Symbol...
⇧⌘M - Show Problems panel
⇧⌘P - Execute command
⌘ + leftClick - SHOW ME DA CODE PART'NA

------ FILE MANAGEMENT: ------

⌘N -  New File
⌘O -  Open File..
⌘S -  Save
⇧⌘S -  Save As...
⌥⌘S -  Save All
⌘W -  Close
⌘K -  ⌘W Close All



WINDOWS

------ NAVIGATION: ------

Ctrl+T - Show all Symbols
Ctrl+G - Go to Line...
Ctrl+P - Go to File...
Ctrl+Shift+O - Go to Symbol...
Ctrl+Shift+M - Show Problems panel
Ctrl + leftClick - SHOW ME DA CODE PART'NA


------ FILE MANAGEMENT: ------

Ctrl+N New File
Ctrl+O Open File...
Ctrl+S Save
Ctrl+Shift+S Save As...
Ctrl+K S Save All
Ctrl+F4 Close
Ctrl+K Ctrl+W Close All
